Writing Your First Program

Here’s how you can write a simple print statement in Python:

print("Hello, World!")

This code will output:

Hello, World!

Working with Variables

In programming, a variable is used to store data. Here’s an example:


name = "Satyam"
age = 20
print(f"My name is {name} and I am {age} years old.")

When you run this program, it will output:

My name is Satyam and I am 20 years old.

Keyboard Shortcuts

When coding, you can use keyboard shortcuts to speed up your workflow. For example:

  • Press Ctrl + S to save your file.
  • Press Ctrl + Z to undo changes.
