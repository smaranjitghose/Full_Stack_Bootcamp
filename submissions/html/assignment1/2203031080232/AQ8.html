Glossary

Programming Terms
Variable
A storage location identified by a memory address and an associated symbolic name (an identifier), which contains some known or unknown quantity of information referred to as a value.

Example: In Python, x = 5 declares a variable x with the value 5.

Function
A block of code designed to perform a specific task. Functions allow code to be reused and organized into modular pieces.

Example: In JavaScript, function add(a, b) { return a + b; } defines a function that returns the sum of two numbers.

Loop
A control structure that repeats a block of code as long as a specified condition is true. Common types include for loops and while loops.

Example: A for loop in C++ might look like for(int i = 0; i < 5; i++) { cout << i; }.

Web Development
HTML
The standard markup language used to create web pages. It structures the content on the web.

Example: The basic structure of an HTML document:

<!DOCTYPE html>
<html>
<head></head>
<body></body>
</html>
Computer Science Concepts
Algorithm
A step-by-step procedure for solving a problem or performing a task. Algorithms are used in computer programs to perform operations and make decisions.

Example: The steps to sort an array using the bubble sort algorithm.

Data Structure
A specialized format for organizing and storing data. Common types include arrays, linked lists, stacks, and queues.

Example: A stack follows a Last In First Out (LIFO) principle.

Object-Oriented Programming (OOP)
A programming paradigm based on the concept of "objects," which can contain data in the form of fields and code in the form of procedures.

Example: Defining a class in Python:

class Dog:
  def __init__(self, name):
    self.name = name
  def bark(self):
    print("Woof!")
