Glossary of Terms

Web Development

HTML
HyperText Markup Language - the standard markup language for web pages.
CSS
Cascading Style Sheets - A style sheet language used for describing the presentation of a document written in HTML.
JavaScript
A programming language used to create dynamic content on websites, such as interactive forms and animations.
Responsive Design
A design approach that makes web pages render well on a variety of devices and window or screen sizes.

Programming Languages

Python
A high-level, interpreted programming language known for its ease of use and readability.
Java
A class-based, object-oriented programming language that is designed to have as few implementation dependencies as possible.
C++
A general-purpose programming language that extends C with object-oriented features, often used for system/application software.
Ruby
A dynamic, object-oriented programming language known for its simplicity and productivity.

Additional Concepts

Cloud Computing
Using remote servers hosted on the Internet to store, manage, and process data, instead of a local server or personal computer.
API
Application Programming Interface - a set of rules that allows one software program to interact with another.
Version Control
A system that records changes to files, allowing you to track modifications and collaborate with others.
