Glossary

Web Development Terms

HTML
HyperText Markup Language, the foundation of web pages.
Used to create structure and content for websites.
CSS
Cascading Style Sheets, used to style web pages.
Controls layout, colors, and fonts.
JavaScript
A programming language for creating interactive web elements.
Examples include animations and form validations.

Related Terms

Frontend
The visible part of a website that users interact with.
Backend
The server-side logic that handles data and processing.
